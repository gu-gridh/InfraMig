# InfraMig

An interactive web map for exploring worker distribution to SSAB and Stegra, by country of origin, industry branch, and time.

Built with Vue 3, Leaflet, and Vuetify, this application visualizes geospatial data from GeoJSON layers and provides dynamic filtering and temporal analysis.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
